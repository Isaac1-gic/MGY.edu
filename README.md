# MGY (Malawian Genius Youths) 🚀
**A full learning and networking hub for tech and innovation.**

MGY is a Progressive Web App (PWA) designed to empower the youth of Malawi by providing a central platform for science, education, and collaborative innovation. It bridges the gap between learning and networking, allowing students and innovators to share knowledge, access quality courses, and build professional identities.

[Live Demo](https://mgy265.netlify.app) | [Backend API](https://mgy265.onrender.com)

---

## 🌟 Key Features

- **📚 Educational Courses:** A curated library of learning materials focusing on science and technology.
- **💬 MGY Room:** A real-time collaborative space for discussions, quizzes, and technical troubleshooting.
- **👤 User Profiles:** Personalized networking hubs where members can showcase achievements and build their local tech network.
- **📲 Offline First:** Built as a PWA with service worker support, ensuring that learning continues even with intermittent internet connectivity.
- **🔒 Secure Authentication:** A custom-built authentication flow using Python backends and Firebase Custom Tokens for universal device compatibility.

---

## 🛠️ Tech Stack

### Frontend
- **HTML5 / CSS3:** Responsive design optimized for mobile devices (1.5GB RAM+ friendly).
- **JavaScript (Vanilla):** Light-weight logic for fast performance.
- **PWA Manifest & Service Workers:** For installability and offline capabilities.

### Backend & Database
- **Python (Flask):** Hosted on **Render.com**, handling custom authentication logic and API requests.
- **Firebase Realtime Database:** For high-speed data syncing in the MGY Room.
- **Firebase Auth:** Utilizing Custom Tokens for a seamless login experience across all mobile browsers.
